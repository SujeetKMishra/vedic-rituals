# Pooja Path

A static website for Pooja Path, offering traditional Vedic rituals and poojas at home in Stockholm, Sweden.

## Description

This project is a responsive HTML website built with Tailwind CSS, showcasing services like Satyanarayan Katha, Grihapravesh, Rudrabhishek, and more. It includes a hero section, services grid, contact information, and a scroll-to-top feature.

## Features

- Responsive design for mobile, tablet, and desktop
- Smooth scrolling navigation
- Service cards with hover effects
- Contact via WhatsApp
- Scroll-to-top button
- Multi-language support (English, Hindi, Swedish)

## Language Support

The website supports multiple languages:
- English (default)
- Hindi
- Swedish

Language selection is available via flag icons in the navigation bar.

## How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Font Awesome (via CDN)
- JavaScript (vanilla, for scroll functionality and language switching)
